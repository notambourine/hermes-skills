# Output format (1-3-1)

Emit exactly this shape — one headline, three bullets, one "so what":

```
**<headline — the single claim, ≤12 words>**

- <key point 1>
- <key point 2>
- <key point 3>

**So what:** <one line on why it matters to the reader>
```

Notes:
- The headline is a *claim*, not a topic label. "Fed cut rates 0.5%" beats "About interest rates".
- Bullets are facts that support or qualify the headline, not a table of contents.
- "So what" is the reader's takeaway — the action or implication, in one line.
